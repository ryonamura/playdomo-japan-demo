## Phase 0: Go/No-Go判断（0〜30日）— 予算: 200〜400万円

> **目的**: 体験型モデルの法的実現可能性を確認し、DOMO本社と方針合意する

### Week 1: 情報収集・弁護士選定

- [ ] **DOMO本社から正式な情報開示を受ける**
  - 現行サービスの詳細仕様書（パック構成・Cosmos・抽選メカニズム）
  - 現在の賞品一覧と実績（ランボルギーニ Veneno、Rolex Batman、Tissot PRX Auto）
  - フェラーリ撤去の経緯（参加者不足による中止の詳細）
  - Cosmos通貨のフロー図（取得→保有→消費→当選）
  - フランスでのJONUM規制への対応方針
  - 無料郵送参加（最大10トークン）の実績データ
  - 現金代替オプション（参加規約記載）の過去の適用実績
  - Akatsuki Inc.（東証上場・日本本社）の関与度と意思決定フロー
- [ ] **IT/ゲーム専門弁護士の選定**
  - 候補: TMI総合法律事務所、森・濱田松本、西村あさひ等
  - 必須条件: ガチャ/オリパ/賭博法に実績あり + 体験型モデルの適法性評価可能
  - 初回面談で本レポートのPart 2（法規制分析）を共有
- [ ] **DOMOアプリのフランス版を実体験**
  - VPN経由でフランスApp Storeからダウンロード
  - パック購入→カード排出→Cosmos獲得→アクティベーションのフロー確認
  - 「1 COSMOS = 1 CHANCE」セクションのUI/UX記録
  - 決済フロー確認（Nuvei経由）

### Week 2-3: リーガルオピニオン取得

- [ ] **弁護士に以下の法的判断を正式依頼**
  - 賭博罪（刑法185条）: 体験型賞品が「財産上の利益」に該当するか
  - 富くじ罪（刑法187条）: 体験型賞品での4要件該当性
  - 景品表示法: 体験の「景品類の価額」算定方法と上限（10万円）該当性
  - 風営法: 体験型モデルでの該当性
  - 資金決済法: Cosmos通貨が前払式支払手段に該当するか
- [ ] **体験型賞品の具体例で個別に適法性確認**
  - 美容医療プログラム（50〜200万円相当）
  - 有名人との体験（YouTuber出演、声優レッスン等）
  - ミシュラン三ツ星ディナー
  - ラグジュアリー体験（遊覧飛行、スーパーカードライブ）
  - 成果物が残る体験のリスク（漫画家のオリジナル作品等）
- [ ] **公式サイトの賭博表現に関する法的見解取得**
  - 英語版「Place your bet」/ 仏語版「misez」の日本での訴訟リスク
  - 日本版で全ギャンブル表現を排除した場合の効果

### Week 3-4: Go/No-Go判断

- [ ] **リーガルオピニオンに基づくGo/No-Go判断**
- [ ] **DOMO本社向け提案書を作成・提出**
  - 現行モデルの刑事罰リスク（富くじ罪: 懲役2年 / 賭博罪 / 景品表示法違反）
  - 「Place your bet」表現の訴訟証拠リスク
  - 現金代替オプションの「財物の得喪」強化リスク
  - 体験型モデルへの転換提案
  - 日本版パック価格設計（上限5,000円）
  - 体験賞品25案のラインナップ
  - Phase 1以降のロードマップ・予算
- [ ] **DOMO本社との方針協議**
  - 体験型モデルへの転換合意
  - パック価格変更の承認
  - 日本展開の投資判断
  - Akatsuki Inc. コンプライアンス部門の承認確認

### Go/No-Go 判定パターン

| 結果 | 次のアクション |
|------|----------------|
| **Go**: 体験型モデルで展開可 | → Phase 1へ進む |
| **条件付きGo**: 一部体験のみ可 | → 適法カテゴリに限定してPhase 1開始 |
| **No-Go**: 体験型でもNG | → 純粋デジタルTCG化を検討 or プロジェクト終了 |
